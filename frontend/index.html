Despliegue de Infraestructura Web Segura

Autor: Imad

Descripción: Arquitectura basada en contenedores Docker, Proxy Inverso (Apache), balanceo de carga, seguridad SSL (Let's Encrypt) y autenticación centralizada.

Servicios configurados:

  • Servidor de Aplicaciones 1: Entorno PHP
  • Servidor de Aplicaciones 2: Entorno Java (Tomcat)
  • Área Restringida (Requiere autenticación LDAP en subred privada)
